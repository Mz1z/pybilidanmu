### 项目环境

```bash
需要的python版本>=3.7
pip install websockets            # 用于websocket连接
pip install brotli                # 用于解压数据包
```



### 项目结构

```bash
pybilidanmu.py        # 主要的应用层交互类(完善中)
pybilidanmu_window.py    # 图形化界面(开发中)
```



### 运行方式

```bash
python3 pybilidanmu.py           # 命令行启动
or
python3 pybilidanmu_window.py        # 图形化启动
```
